# COP509 Coursework Dataset: Government Inquiry Recommendations and Responses

This repository contains datasets for the **COP509: Natural Language Processing** coursework at Loughborough University.

## Overview

The dataset comprises pairs of publicly available UK government inquiry documents:
- **Recommendations**: Extracted recommendations from official government inquiry reports
- **Responses**: The government's official responses to those recommendations

## Dataset Contents

| Inquiry | Recommendations | Response |
|---------|-----------------|----------|
| Women's State Pension Age | `governments_new_decision_womens_state_pension_age_communications-Recomm.pdf` | `governments_new_decision_womens_state_pension_age_communications-Response.pdf` |
| Space Economy | `TheSpaceEconomyReport.pdf` | `TheSpaceEconomyResponse.pdf` |
| Behaviour Change | `Behaviour-Change-Report-Recomm.pdf` | `Behaviour-Change-Response.pdf` |
| Infected Blood Inquiry | `Volume_1-Blood-Inquiry-Recomm.pdf` | `Volume_1-Blood-Inquiry-Response.pdf` |
| UK Covid-19 Inquiry (Module 1) | `UK-Covid-19-Inquiry-Module-1-Recomm.pdf` | `UK-Covid-19_Inquiry_Module_1_Response.pdf` |
| Post Office Horizon IT Inquiry | `PostOfficeHorizon-I-Inquiry-Recomm.pdf` | `PostOfficeHorizon-IT-Inquiry-Response.pdf` |

## Intended Use

This dataset is designed for the COP509 coursework.

## Data Pairing Notes (supportive info)

- Recommendations and Responses from each document pair have been aligned using Claude AI (therefore there may be alignment errors). This was  done for convenience to provide a starting point for the coursework).
